What is RoboCasa365?

RoboCasa365 is a simulation benchmark designed to train and evaluate generalist robot policies for everyday household tasks. It focuses on a kitchen robot that must combine navigation, manipulation, and long-horizon planning.

Core idea: scale robot learning by providing diverse, realistic, long-horizon kitchen tasks with rich demonstrations.

Task Taxonomy: 365 Tasks

Atomic Tasks — 65

Single-skill, single-step tasks across 14 fixture types

• Navigation: NavigateKitchen
• Pick & Place: 20 variants (counter↔fridge, sink, oven, microwave…)
• Door/Drawer: Open/Close for 6 appliance types
• Appliance control: Stove knobs, sink faucets, blender lid, toaster, coffee machine…
~100 demos per task
Key insight: atomic tasks are the building blocks that compose into long-horizon tasks. A robot must master all atomic skills to succeed at composite tasks.

Composite Tasks — 300

Multi-step long-horizon tasks spanning 60 activity categories

• Sequences of 2–10+ atomic actions
• Require planning & sequential reasoning
• Example: MakeBananaMilkshake = open fridge → pick banana → place in blender → close lid → turn on blender…
~100 demos per task (human + synthetic)

Navigation: Yes — Mobile Manipulation

RoboCasa features a Franka Panda arm mounted on a mobile base. The robot can navigate between kitchen fixtures, making this a full mobile manipulation benchmark.

NavigateKitchen (atomic task)

Dedicated task for moving between kitchen locations (counter, fridge, sink, stove, microwave, etc.)

Navigation embedded in composite tasks

  • Robot starts near one fixture, must navigate to another to pick up/place objects
  • Multi-fixture tasks (e.g., fridge → counter → stove → sink) require 3–5 navigations per episode
  • Kitchen layouts vary across 2,500+ scenes — no fixed path memorization
Navigation + manipulation = full household robot paradigm, unlike tabletop-only benchmarks.

RoboCasa vs. LIBERO-Long — Deep Dive

LIBERO-Long

  • 10 tasks, each with 4–6 stages of sequential manipulation
  • Episodes: ~60–120 seconds
  • ~50 demonstrations per task
  • Fixed tabletop, no navigation
  • Small visual variety (<10 scenes)
  • All tasks are pre-defined sequences: pick A → place B → pick C → …
  • Human success rate baseline: ~70–80%
LIBERO-Long is an important baseline but limited to 10 tasks, no navigation, small data scale, and no generalization split.

RoboCasa365 Composite Tasks

  • 300 composite tasks across 60 activity categories
  • Episodes: 10–166 seconds (avg 52s, max similar to LIBERO-Long)
  • 100 demonstrations per task (human-collected)
  • Navigation between fixtures embedded in tasks
  • 2,500+ diverse scenes for training; 10 held-out for test
  • Tasks require semantic reasoning (e.g., FIFO fridge loading, size-based sorting, type-based categorization)
  • Best model (GR00T) achieves only 9.6% on composite tasks — much harder
Key gap filled: RoboCasa365 provides the scale, diversity, and difficulty that LIBERO-Long lacks — while being directly comparable in episode length for the shorter-horizon tasks.

Why RoboCasa365 for Long-Horizon Reasoning?

1. Sequential Decision Chains

Tasks require robots to execute 2–10+ sub-goals in the correct order. Failure at any step cascades — unlike single-step benchmarks, there is no partial credit from atomic success alone.

2. Semantic Reasoning Required

Tasks like LoadFridgeFifo, BeverageSorting, or FilterMicrowavableItem require the robot to understand categories, not just mimic motor patterns.

3. Planning Under Partial Observability

Kitchen drawers, fridge compartments, and cabinet interiors are hidden. The robot must plan retrieval sequences without full state visibility — mirroring real-world challenges.

4. Temporal Extended Actions

Many tasks span 10–166 seconds at 20 Hz control = 200–3,320 control steps per episode. This directly tests long temporal credit assignment in policy learning.

5. Task Composition & Generalization

Composite-Unseen tasks in evaluation test whether learned skills compose to novel tasks — a key measure of genuine planning vs. memorization.

6. Real-World Action Space

Appliance control (stove knobs, faucet handles, blender buttons) + navigation + dexterous manipulation = the full action space of household robots.

Three Benchmarking Protocols

1. Multi-Task Learning

Goal: Learn a single policy across all 300 tasks from 482 hrs of human data.

Train: Human demos, 300 tasks, 100 demos/task

Eval: Pretrain scenes

Measures: Breadth of multi-task generalization

2. Foundation Model Learning

Goal: Pretrain on 2,200+ hrs, fine-tune on 50 target tasks.

Pretrain: 482 hrs human + 1,615 hrs synthetic

Fine-tune: 193 hrs, 3 splits (atomic/comp seen/unseen)

Eval: Held-out target scenes

Measures: Transfer, data efficiency

3. Lifelong Learning

Goal: Sequentially learn harder tasks without forgetting.

4 phases:

  • Phase 1: Atomic tasks (~41.5% success)
  • Phase 2: 2–3 stage tasks (~24.5%)
  • Phase 3: 4–5 stage tasks
  • Phase 4: 6+ stage tasks

Measures: Continual learning, forgetting

Supported algorithms: Diffusion Policy, π₀ / π₀.₅ (OpenPI), GR00T N1.5 — representing the current frontier of visuomotor robot learning.
